# back to basics

2.7년차 백엔드 개발자의 실력 재정비 학습 저장소. 2027 이직 목표.

목표는 지식량이 아니라 **원리·트레이드오프의 언어화**. "봤다"가 아니라 다음 학습 사이클로 체득한다.

## 학습 사이클 (모든 강의 완료 정의)

1. 학습 — 강의 한 단위 시청
2. 액티브 리콜 — 덮고 백지에 핵심 재현
3. 핸즈온 재현 — 직접 코드로 짜고 변형·실험 (← 이 저장소)
4. 파인만/TIL — [velog](https://velog.io/@kdelay)에 "남이 이해하게" 1편 작성
5. "왜?" — 트레이드오프 1개 기록

## 주간 목표

- 주 3강 + TIL 2편
- 평일 저녁 9~10시 고정

## 구조

Phase별 디렉토리에 핸즈온 코드를, `qna/`에 학습 QnA 원본을 쌓는다. (Phase 디렉토리는 해당 Phase 진행 시 생성)

```
phase1-web-spring/         # P1 웹·스프링 기초
phase2-db-jpa/             # P2 DB·JPA 기초
phase3-internals/          # P3 스프링 내부 원리
phase4-jpa-practice/       # P4 JPA 실전·성능
phase5-cs-basics/          # P5 CS 기초 (병행)
phase6-backend-practical/  # P6 백엔드 실전 (동시성·Redis·Security·MQ·테스트)
phase7-system-deep/        # P7 시스템 내부 심화 (JVM·DB·네트워크·OOP)
phase8-design-system/      # P8 설계·시스템 디자인 + 기술 블로그
phase9-coding-test/        # P9 코딩테스트 (병행)
qna/                       # 학습 QnA 원본 → velog·전자책 소스 (qna/README.md)
```

## 진행 상황

전체 로드맵(Phase 0~10)·이직 최소 경로·이슈는 [Linear](https://linear.app/klilly/project/백엔드-실력-다지기-이직-준비-2027-950ee5d4b255)에서 관리한다. **필수 경로만 밟아도 이직 지원, 심화(P7)까지 채우면 완전 마스터**가 목표.
